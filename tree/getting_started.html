Installing py-pilecore

To install pypilecore, we strongly recommend using Python Package Index (PyPI). You can install pypilecore with:

pip install pypilecore

How to import py-pilecore

Getting started with pypilecore is easy done by importing the pypilecore library:

In [1]: import pypilecore

or any equivalent import statement.
